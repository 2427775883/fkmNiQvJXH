# 项目简介

## 核心功能
- 登录功能
- 注册功能
- 学院信息管理
- 专业信息管理
- 班级信息管理
- 课程信息管理
- 学生信息管理
- 教师信息管理
- 选课信息管理
- 信息查询

## 适用场景
- 教育机构内部学生选课管理
- 学生个人信息管理
- 教师课程信息管理
- 课程选课信息跟踪

## 技术特点
- 使用 Python 和 Django 框架构建后端
- 前端技术包括 HTML、CSS、JavaScript、jQuery
- 数据库采用 MySQL
- 支持Vscode、navicat等开发工具
- 运行环境要求 Python3.0 以上、MySQL 8.0
